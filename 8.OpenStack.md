## 03. Cloud Computing $ Openstack

- on-premise vs cloud

-  클라우드 컴퓨팅이란: 사용자 요청에 따라 공유된 자원이나 데이터를 인터넷 기반으로 제공하는 기술로, 여러 유형의 서비스를 사용한 만큼 지불하는 종량과금제로 제공되는 computing 환경

  

- 클라우드 서비스 유형

  - IaaS

  - PaaS

  - SaaS

    

- 클라우드 배치 유형

  - public
  - private
  - hybrid
  - community Cloud



- 오픈스택이란 == Cloud OS



- Core Service of Open Stack 이해

  - compute

  - image

  - object storage

  - block storage

  - network

  - dashboard

  - identity

  - orchestration
